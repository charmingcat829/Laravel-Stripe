# LaravelStripe
Laravel Stripe payment

# Installation

```bash
composer require code4mk/lara-stripe
```

# Setup

## vendor publish

```bash
php artisan vendor:publish --provider="Code4mk\LaraStripe\LaraStripeServiceProvider" --tag=config
```
